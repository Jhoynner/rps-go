Acerca de la Aplicacion

Esta aplicacion web del juego Piedra, Papel o Tijera fue desarrollada con el lenguaje de programacion GO, utilizando tambien JAVASCRIPT, HTML y CSS.

Se desarrollo con el objetivo de ir aprendiendo cada vez mas sobre este lenguaje de una forma didactica como lo es el famoso juego de Piedra Papel o Tijera.
